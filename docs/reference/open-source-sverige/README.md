# Open Source i svensk offentlig sektor — Referenser

> Källor för att styrka positioneringen av dearbetarfordig.se som open source civic tech.

---

## 1. offentligkod.se

Huvudkatalog för open source-lösningar i svensk offentlig sektor.

- **URL:** https://offentligkod.se
- **Vad:** Listar konkreta open source-verktyg som används av svenska myndigheter och kommuner (Element, Alfresco, Angular-baserade lösningar m.fl.)
- **Användning:** Direkt citering möjlig. Visar att det finns en rörelse mot öppen kod i svensk förvaltning.
- **Drivs av:** NOSAD

---

## 2. DIGG Sweden (GitHub)

Officiell GitHub för Myndigheten för digital förvaltning.

- **URL:** https://github.com/diggsweden
- **Vad:** Interna riktlinjer för open source — "Handbok för arbete med öppen programvara"
- **OBS:** Intern policy, inte obligatoriskt krav. Formuleringen ska vara exakt.
- **Användning:** Visar att statliga myndigheter aktivt arbetar med open source-riktlinjer.

---

## 3. Riksdagen — Motion 2025/26:315 + Betänkande FiU25

Riksdagsbeslut om tillkännagivande till regeringen om övergång till open source.

- **Motion:** 2025/26:315 (Mattias Bäckström Johansson, SD)
- **Betänkande:** 2025/26:FiU25
- **Nyckeltal:** 60% av svenska kommuner saknar strategi för open source
- **Internationell referens:** Nederländerna sparade 45 mnkr genom öppna standarder i kadaster
- **Status:** Riksdagen stödde tillkännagivande — regeringen ska agera
- **Användning:** Starkaste politiska argumentet. Riksdagen har tagit ställning.

---

## 4. Försäkringskassan — open source-initiativ

Konkreta exempel på statlig myndighet som öppnar sin kod.

- **URL:** https://forsakringskassan.se (pressmeddelanden 2024-2025)
- **Designsystem:** Öppnades som open source (oktober 2024)
- **IT-drift:** Mandat för samordnad statlig IT-drift för 70+ myndigheter (1 januari 2025)
- **Användning:** Bevisar att trenden är reell — inte bara policy utan implementering.

---

## 5. NOSAD

Nätverket för öppen och samverkande digital förvaltning.

- **URL:** Via offentligkod.se
- **Vad:** Organisationen bakom offentligkod.se. Kontaktpunkt för svenska open source-initiativ i offentlig sektor.
- **Användning:** Rätt ingång för samarbete/kontakt. Inte Kivos eller Open Jämtland.

---

## 6. Open Source Sweden

Branschorganisation för open source i Sverige.

- **URL:** https://opensourcesweden.se
- **Vad:** Kan vara källa till statistik eller expertutlåtanden om open source i Sverige.
- **Användning:** Expertröst vid behov (media, rapporter).

---

## Relevans för dearbetarfordig.se

Dessa källor styrker vår positionering:

1. **Vi följer en nationell trend** — Riksdagen har beslutat om öppen kod i offentlig sektor
2. **Vi fyller ett gap** — 60% av kommuner saknar strategi → vi ger dem ett verktyg
3. **Vi är i gott sällskap** — Försäkringskassan, DIGG, NOSAD driver samma riktning
4. **Vi är oberoende** — AGPL-3.0, self-hosted, inga US-cloud-beroenden
